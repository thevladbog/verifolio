# Перед началом: согласие на обработку данных

Чтобы собрать ваш ответ, Verifolio обработает ваше имя, email и отправленный вами контент и сохранит их в регионе этого развёртывания. Ответ передаётся запросившему и — только с вашего явного разрешения по каждому файлу — показывается на публичной странице проверки. Согласие можно отозвать позже; отзыв останавливает дальнейшую обработку.
